# 桌面圖片

* [設定片段](https://github.com/samwhelp/note-about-xfce/blob/gh-pages/_demo/config/xfce-config/main/config/xfce4/xfconf/xfce-perchannel-xml/xfce4-keyboard-shortcuts.xml#L89)

| 按鍵組合         | 功能                 | 執行指令                                         |
| ---------------- | -------------------- | ------------------------------------------------ |
| `Alt + w`        | 隨機更換新的桌面圖片 | `wallpaper-select-xfce shuf_favorite` |
| `Alt + Ctrl + w` | 更換成預設的桌面圖片 | `wallpaper-select-xfce default` |
